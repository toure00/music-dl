* 音乐下载器
用于某云音乐下载，测试环境 Linux，其他操作系统未测试。

** 依赖
- [[https://github.com/babashka/babashka][babashka]]
- [[https://ffmpeg.org/][FFmpeg]]

** 上游 api
[[https://github.com/Suxiaoqinx/Netease_url][Netease_url]]

** 使用方式
安装所需工具后运行
#+begin_src shell
❯ bb music-dl
{:path ./, :l lossless, :dl hard}
Usage:
  -pid, Play list ID
  -aid, Album ID
  -sid, Song ID
  -qs,  Query song name
  -ql,  Query song list

Optional:
  -dl,  Download lyric mode
  -l,   Quality level
#+end_src

** 配置文件
配置文件 [[./config.yaml][config.yaml]] 和脚本在同一层目录下，
- api (api 参数)
  + v*    (api host version)
  + info  (歌曲信息api)
  + album (专辑api)
  + list  (歌单api)
  + lyric (歌词api)
  + durl  (歌曲下载链接api)

- opt (设置默认可选参数)
  + api   (选择 api 版本)
  + level (音质等级)
  + lyric (歌词格式，hard[内嵌], ext[外挂])
